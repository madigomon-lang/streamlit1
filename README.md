Este proyecto ha sido desarrollado como parte del curso de **Análisis de Datos de Talento Tech (Nivel Integrador)**. La aplicación utiliza el dataset de clasificación de materiales de reciclaje de Chile para proporcionar insights estratégicos sobre la gestión de residuos y materiales peligrosos.

## 👤 Autoría
Realizado por: **Maria Dilia** *Experto en Análisis de Datos*

## 🚀 Características
* **Landing Page Informativa:** Explicación detallada del dataset y contexto ambiental con interfaz moderna.
* **Panel de Trabajo Profesional:** Uso de KPIs y gráficos avanzados para la toma de decisiones.
* **Análisis Multivariable:** Gráficos de Seaborn (Boxplots, Regplots) y Plotly (Barras dinámicas).
* **Validación de Modelos:** Sección dedicada a validar la regresión lineal (R², MSE, análisis de residuos).
* **Documentación Interactiva:** Información contextual mediante pestañas e iconos de ayuda.

## 🛠️ Requerimientos Técnicos
- Python 3.8+
- Librerías: `streamlit`, `pandas`, `numpy`, `matplotlib`, `seaborn`, `plotly`, `scikit-learn`.
